## Innovation Project Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/31b2456dafbc4ba3b6215adb0adf50a5/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, supervising teachers, department administrators, and administrators.

**Regular User Features:**
1. Login
2. Register
3. Homepage
4. Excellent Projects
5. Project Information
6. Review Plans
7. Innovation News
8. Personal Center

**Supervising Teacher Features:**
1. Login
2. Register
3. Personal Center (Change Password, Personal Information)
4. Excellent Project Management
5. Project Type Management
6. Project Information Management
7. Project Application Management
8. Project Mid-term Review Management
9. Project Completion Management
10. Project Review Management
11. Expert Review Management
12. Review Plan Management

**Department Administrator Features:**
1. Login
2. Register
3. Personal Center (Change Password, Personal Information)
4. Excellent Project Management
5. Project Type Management
6. Project Information Management
7. Project Application Management
8. Project Mid-term Review Management
9. Project Completion Management
10. Project Review Management
11. Expert Review Management
12. Review Plan Management




**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Student Management
4. Supervising Teacher Management
5. Department Administrator Management
6. Excellent Project Management
7. Project Type Management
8. Project Information Management
9. Project Application Management
10. Project Mid-term Review Management
11. Project Completion Management
12. Project Review Management
13. Expert Review Management
14. Review Plan Management
15. System Management (Innovation News, Carousel Management)
